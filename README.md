Prácticas completas de la asignatura Programación y Diseño Orientado a Objetos (PDOO) del curso 23/24.

Se implementa un juego completo (irrgarten) con interfaz gráfica incluida. 
El foco de esta implementación es el diseño orientado a objetos. Para ello se implementa una serie de clases siguiendo una organización adecuada para este tipo de diseño y que se adecuan a los diagramas proporcionados por los profesores.

Lenguaje de programación: Java.
Nota final de prácticas: 10/10.
